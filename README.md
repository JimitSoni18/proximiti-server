# proximiti-server
An place to hang out and have fun!

## Goal

> Provide a self hostable social platform that is

- Fast to run on legacy hardware
- Efficient to run on any hardware
- Secure by ensuring high level authentication and acccess control mechanisms

## Milestones

### Before v1.0

[] Basic HTTP REST server that is able to receive requests and send responses
[] Integration with a testing database (Most probably SQLITE)
[] API testing
[] Designing a minimal Front-end to perform basic crud operations (WEB - SOLID JS)
[] Add websocket communication for instant messaging
[] Specification for federation between servers

### After v1.0

[] Design micro-service API for different components
[] Transition to GraphQL if need be
[] Implementation of Role/Attribute Based Access control
[] Transition to using Protocol buffers for efficiency
[] Make client for android in Kotlin
[] Native client for desktop OS (Rust)
[] Bring support for real time voice and video exchange using WebRTC rust

### Later during the journey

[] Make a native iOS client using Objective-C + Swift
